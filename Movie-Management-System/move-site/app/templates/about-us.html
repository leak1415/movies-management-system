Who We Are?

We are a team of students building a simple web application to manage and display movie information. This project helps us understand the fundamentals of Flask, a lightweight web framework for Python.

History

We started this project to learn Flask, databases, and web development basics. Our goal was to create a fully functional movie management system.

Value

Our application demonstrates CRUD operations, filtering, searching, and exporting data, giving practical experience with Python and MySQL.

Mission

To build an educational project that allows students to interact with a simple movie database and understand web application development.
